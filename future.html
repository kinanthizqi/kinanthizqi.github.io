Rencana Saya di Masa Depan

Kalau ditanyakan mengenai rencana saya ke depannya, saya belum berpikir terlalu jauh. Saya rasa saya akan terus menjadikan modelling menjadi karir saya untuk beberapa tahun di masa depan karena saya sangat senang menjadikan modelling sebagai pekerjaan sehari-hari. Walau terkadang letih, saya tetap senang berpose di depan kamera ataupun berjalan di fashion show musiman setiap tahunnya.

Saya sangat ingin suatu saat saya bisa dipilih untuk kampanye mode yang terpampang jelas di jalanan besar di Kota Paris. Saya bukan orang yang suka dilihat dimana-mana, tetapi projek kampanye mode yang besar bersama brand fenomenal untuk sesekali saja mungkin menyenangkan. Untuk merealisasikan hal tersebut, saya berencana untuk berusaha lebih keras dan lebih terbuka terhadap kesempatan-kesempatan yang diberikan. Mungkin saya akan lebih giat lagi jalan di runway fashion show berbagai merk, baik brand besar maupun yang belum dikenal masyarakat luas. Saya berencana untuk mematokkan "walk" saya per musim di sebanyak 38 merk. Agak sedikit lebih banyak dibandingkan yang saya kerjakan sebelumnya.

Dan jikalau nantinya saya ingin melakukan hal lain di masa depan, ada kemungkinan bagi saya untuk membuat sebuah brand pakaian hasil rancangan saya sendiri atau membuka exhibition hasil jepretan saya. Saya juga berpikiran untuk bisa mencoba akting serial televisi ataupun film layar lebar. Namun, mungkin itu rencana di lain hari nanti. Untuk sekarang dan beberapa tahun ke depan saya masih ingin bekerja sebagai seorang model profesional.
